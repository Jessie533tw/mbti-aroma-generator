# MBTI × 芳療香氛配方產生器

- **雙重選擇模式**：直接選擇 MBTI 類型或通過簡單測驗獲得結果
- **16 型完整資料**：每種 MBTI 類型都有專屬香氛配方

- **匯出功能**：可下載配方卡為 PNG 或 PDF 格式

## 🎯 使用方式

1. 開啟 `index.html` 文件
